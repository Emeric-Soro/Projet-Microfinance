================================================================================
 INSTRUCTIONS D'EXECUTION
================================================================================

 Pour executer TOUS les tests (avec Oracle XE container):
   cd backend
   mvn clean test

 Pour executer uniquement les tests unitaires (sans base de donnees):
   mvn test -Dtest="!*IntegrationTest"

 Pour executer les tests d'integration avec Testcontainers:
   mvn test -Dtest="*IntegrationTest"

 Pour un controller specifique:
   mvn test -Dtest="ClientControllerTest"

 Pour les workflows:
   mvn test -Dtest="*WorkflowTest"

 Pour les tests de securite:
   mvn test -Dtest="*Security*,*JwtToken*"

 Pour les tests comptables extremes:
   mvn test -Dtest="ExtremeAccountingTest"

 Pour les tests de concurrence:
   mvn test -Dtest="ConcurrencyConflictTest,IdempotencyTest"
